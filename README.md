# Reduced World Infos
https://modrinth.com/mod/reduced-world-infos

_**Reduced World Infos**_ is a gameplay-focused mod that removes most of the information from Minecraft’s debug screen (F3) and replaces it with immersive, craftable tools.

Instead of accessing everything instantly, players must now rely on specialized items to gather information about their surroundings—making exploration more intentional and rewarding.

![Reduced World Infos Items](https://cdn.modrinth.com/data/cached_images/7b11c3fa22523f57d479e470299aab717c24361d.png)

## Features
### Minimal Debug Screen

The F3 menu is stripped down, removing most informational clutter to encourage a more immersive experience.

### Craftable Information Tools

Access important data through unique items:\
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; - **Altimeter**: Displays your current Y-level (height).\
![Altimeter Recipe](https://cdn.modrinth.com/data/cached_images/fb7daf3a6fecaabfe0b37ecd08d32d0ef98db7b8.png)\
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; - **Attuned Clock**: Shows how many days have passed since the world was created.\
![Attuned Clock Recipe](https://cdn.modrinth.com/data/cached_images/6e04436caac32d5ca9df69c6b064fdfb0bc933ae.png)\
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; - **Biome Recognizer**: Identifies the biome you are currently in.\
![Biome Recognizer Recipe](https://cdn.modrinth.com/data/cached_images/7335c871c9eb0cd9d3848b4ad7190c4fad7e59f6.png)\
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; - **Light Meter**: Reveals the light level of the block you’re standing on.\
![Light Meter Recipe](https://cdn.modrinth.com/data/cached_images/bd699d86c76cefeee4fe63f8f7a441fba5b89df0.png)\
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; - **Logbook**: Tracks your position using relative coordinates based on a customizable origin point.\
![Logbook Recipe](https://cdn.modrinth.com/data/cached_images/8322002adee70603d602a0bad103d65bd91f717d.png)\
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; - **Calibration Shard**: An upgrade item that allows the Logbook to display global coordinates.\
![Calibration Shard Recipe](https://cdn.modrinth.com/data/cached_images/99cc897fe3058965982fdcf9bf9d8515f6e90e9e.png)
### Upgradable Logbook System

The Logbook can be combined with the other tools to consolidate all information into a single, versatile item.

How to combine items with the Logbook:
Hold the Logbook in one hand and the item in the other hand, then right-click to combine them.

**Logbook controls:**

- Right-click (hold for 3 seconds): Set the origin for relative coordinates.
- Shift + Right-click: Change the Logbook mode.
### Reworked Compass

The vanilla compass no longer points to spawn. Instead, it always points north and displays cardinal directions, making navigation more intuitive.

### Enhanced Recovery Compass

The Recovery Compass gains a new ability to detect the nearest mob, even through walls, adding a unique utility beyond its original purpose.
## Design Philosophy

_**Reduced World Infos**_ is designed to:

- Encourage exploration without relying on debug data
- Add progression to information gathering
- Improve immersion while maintaining functionality
